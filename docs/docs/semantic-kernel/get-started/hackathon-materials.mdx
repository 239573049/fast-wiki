# Semantic Kernel的黑客马拉松材料

使用这些材料，您可以举办自己的Semantic Kernel黑客马拉松，这是一个动手操作的活动，您可以在其中使用Semantic Kernel工具和资源学习和创建AI解决方案。

通过参与和举办Semantic Kernel黑客马拉松，您将有机会：

- 探索Semantic Kernel的特性和功能，以及它如何帮助您解决AI相关的问题
- 团队合作，使用Semantic KernelSDK和服务，进行头脑风暴和开发自己的AI插件或应用程序
- 展示您的成果，并从其他参与者那里获得反馈
- 充分享受！

## 下载材料

要举办自己的黑客马拉松，您首先需要下载材料。您可以在此处下载zip文件：

[下载黑客马拉松材料](https://aka.ms/sk/hack/materials)

解压文件后，您将找到以下资源：

- 黑客马拉松样本议程
- 黑客马拉松先决条件
- 黑客马拉松引导者演示
- 黑客马拉松团队模板
- 有用的链接

![黑客马拉松材料](https://learn.microsoft.com/zh-cn/semantic-kernel/media/hackathon-materials.png)

## 准备黑客马拉松

在黑客马拉松之前，您和您的同行需要下载并安装运行Semantic Kernel所需的软件。此外，您应该已经拥有OpenAI或Azure OpenAI的API密钥，并访问Semantic Kernel存储库。请参考引导者材料中的先决条件文档，了解参与者在黑客马拉松之前应完成的任务的完整列表。

您还应该熟悉现有的文档和教程。这将确保您对Semantic Kernel的核心概念和功能有所了解，以便您在黑客马拉松期间帮助他人。强烈推荐以下资源：

- [什么是Semantic Kernel？](https://learn.microsoft.com/zh-cn/semantic-kernel/overview/)
- [Semantic Kernel示例](https://learn.microsoft.com/zh-cn/semantic-kernel/get-started/tutorials)
- [Semantic Kernel示例视频](https://devblogs.microsoft.com/semantic-kernel/recipes/)
- [Semantic KernelLinkedIn培训视频](https://www.linkedin.com/learning/introducing-semantic-kernel-building-ai-based-apps/introducing-semantic-kernel)

## 运行黑客马拉松

黑客马拉松将包括六个主要阶段：欢迎、概述、头脑风暴、开发、展示和反馈。

以下是每个阶段的大致议程和结构，但请根据您的团队随意修改：

| 时长（分钟） | 阶段         | 描述                                                     |
| :----------- | :----------- | :------------------------------------------------------- |
| 15           | 欢迎         | 黑客马拉松引导者将欢迎参与者，介绍黑客马拉松的目标和规则，并回答任何问题。 |
| 60           | 概述         | 引导者将通过预录制的视频或现场演示，向您概述AI的重要性，以及为什么AI对解决当今世界的问题至关重要。除了Semantic Kernel及其特性的概述，如[内核](https://learn.microsoft.com/zh-cn/semantic-kernel/create-chains/kernel)、[规划者](https://learn.microsoft.com/zh-cn/semantic-kernel/create-chains/planner)、[插件](https://learn.microsoft.com/zh-cn/semantic-kernel/create-plugins/)、[记忆](https://learn.microsoft.com/zh-cn/semantic-kernel/memories/)等。您还将看到Semantic Kernel如何用于不同的场景的演示。 |
| 120          | 头脑风暴     | 引导者将帮助您根据您的兴趣或技能水平组建团队。然后，您将使用设计思维技术头脑风暴您自己的AI插件或应用程序的想法。 |
| 360+         | 开发         | 您将使用Semantic KernelSDK工具和资源来开发、测试和部署您的项目。根据可用时间和要解决的问题，这可能需要一整天或多天的时间。 |
| 60           | 展示         | 每个团队将使用提供的PowerPoint模板展示他们的成果。您将有约15分钟的时间来展示您的项目，演示其工作原理，并解释它如何通过AI解决问题。您还将从其他参与者那里获得反馈。 |
| 30           | 反馈         | 每个团队都可以与团队分享他们对黑客马拉松和Semantic Kernel的反馈，并填写[黑客马拉松退出调查](https://aka.ms/sk/hack/feedback)。 |

## 黑客马拉松后的跟进

我们希望您喜欢举办Semantic Kernel黑客马拉松的整个过程！我们很乐意听取您对未来内容的运作情况、不足之处以及改进意见。请花几分钟时间填写[黑客马拉松引导者调查](https://aka.ms/sk/hack/facilitatorfeedback)，与我们分享您的反馈和建议。

- [Semantic KernelDiscord社区](https://aka.ms/sk/discord)
- [Semantic Kernel博客](https://devblogs.microsoft.com/semantic-kernel/)
- [Semantic KernelGitHub存储库](https://github.com/microsoft/semantic-kernel)
